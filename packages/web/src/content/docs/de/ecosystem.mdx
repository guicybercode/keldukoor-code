Eine Sammlung von Community-Projekten, die auf LocalCode basieren.

:::note
Möchten Sie Ihr LocalCode-bezogenes Projekt zu dieser Liste hinzufügen? Senden Sie einen PR.
:::

Sie können sich auch [awesome-localcode](https://github.com/awesome-localcode/awesome-localcode) und [localcode.cafe](https://localcode.cafe) ansehen, eine Community, die das Ökosystem und die Community zusammenfasst.
