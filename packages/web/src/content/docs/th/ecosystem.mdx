ชุดโครงการชุมชนที่สร้างขึ้นบน LocalCode

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ LocalCode ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-localcode](https://github.com/awesome-localcode/awesome-localcode) และ [localcode.cafe](https://localcode.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน

---
